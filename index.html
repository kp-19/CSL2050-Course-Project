Abstract

The objective of this project was to identify and thereby predict the sentiment of short text (tweets) as whether it is negative, neutral or positive. This included implementation of several classical Machine Learning algorithms and techniques, which included:

  • Perceptron
  • Support Vector Machine
  • Decision Trees
  • Ensemble Learning
  • K-nearest neighbour

Also, prior to application of the aforementioned techniques, the dataset itself was processed in the 4 ways listed below:

  • X
  • Y
  • Principal Component Analysis
  • Linear Discriminant Analysis
